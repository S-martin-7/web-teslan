# Teslan — Web UI Kit

A pixel-faithful recreation of the Teslan marketing homepage, componentized into small JSX pieces that compose into the full page (`index.html`).

## Components
- `Header.jsx` — sticky logo + nav + right-side CTA cluster (orange + WhatsApp)
- `Hero.jsx` — full-bleed photo with navy-to-cyan gradient wash, gradient-clipped headline, paired CTAs
- `FeatureBar.jsx` — 4-up dark feature chips that sit under the hero
- `ServicesGrid.jsx` — 4-up service pillar cards with orange accent bars
- `CoverageMap.jsx` — dark section with north-of-Chile city list + subtle map motif
- `ProjectShowcase.jsx` — 3-up photo + caption row
- `CtaBanner.jsx` — closing orange/WhatsApp dual-CTA banner
- `Footer.jsx` — dark footer with logo, link columns, contact block
- `FloatingWhatsApp.jsx` — fixed bottom-right WhatsApp bubble

`index.html` composes every component into a single scrollable page matching the reference mockup.
